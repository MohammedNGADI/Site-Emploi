# Site d'emploi - PFE EST

## Description
Plateforme de recrutement en ligne développée dans le cadre de mon Projet de Fin d'Études (PFE) à l'EST. 
Le site permet aux candidats de consulter et postuler aux offres d’emploi, et aux recruteurs de publier et gérer leurs annonces.

## Fonctionnalités
- Authentification candidats/recruteurs
- Gestion des offres d’emploi (CRUD)
- Postuler en ligne
- Interface responsive

## Technologies utilisées
- Frontend : HTML, CSS, JavaScript
- Backend : PHP
- Base de données : MySQL

## Installation / Lancement
1. Cloner le projet : `git clone https://github.com/username/nom-du-projet.git`
2. Importer la base de données MySQL fournie
3. Configurer le fichier de connexion à la base de données
4. Ouvrir dans un serveur local (XAMPP, WAMP…)
